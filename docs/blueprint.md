# **App Name**: QYMail Replicator

## Core Features:

- Email Parsing: Parse email content (headers, body, attachments) from the provided source (likely .mbox or similar format).
- Database Integration: Store and retrieve email data. Uses local database.
- Email Display: Display parsed email data in a user-friendly format.
- Email Search: Allow users to search through stored emails based on various criteria (sender, recipient, subject, content).
- Attachment Handling: Allow users to download or view attachments.
- User Authentication: Implement user registration, login, and session management, now with Google Authentication.
- AI powered Mail Summarization: Utilize AI to generate a short summary of the emails for quick reading, uses the AI model as a tool, summarizing the purpose and content
- Security Dashboard: Provide a dashboard displaying threat analysis and security-related information, using AI to analyze potential threats.

## Style Guidelines:

- Primary color: A calming, desaturated blue (#6699CC) to evoke a sense of reliability and professionalism.
- Background color: A very light, near-white blue (#F0F8FF) that's easy on the eyes and provides a clean backdrop for email content.
- Accent color: A vibrant, contrasting orange (#FFA07A) for interactive elements and important notifications.
- Body text: 'Inter' (sans-serif) for a modern, readable look. Headline: 'Space Grotesk' for the headings.
- Use clear and simple icons from a consistent set to represent common actions and email attributes (e.g., reply, forward, attachment).
- Employ a clear, well-structured layout that optimizes for readability. Prioritize key email elements (sender, subject, body) and provide intuitive navigation.
- Incorporate subtle animations to enhance the user experience and provide visual feedback during interactions.
